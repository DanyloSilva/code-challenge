# Como executar os arquivos

## Importar no Eclipse
- Abra o Eclipse IDE ;
- Clique em File > Import...;
- Expanda a opção General e escolha a opção Existing Projects into Workspace;
- Marque Select root directory e clique em browse;
- Localize a pasta "./code-challenge/java/bastosgabriel312", clique em open e finish;

## Executar
- Expanda o projeto "code-challenge" ou a pasta src;
- Expanda o pacote referente ao desafio que deseja executar Ex: br.com.desafio1;
- Clique com o direito na classe DesafioMain e clique em Run As > Java application.
