Importante salientar que, Python não vem com suporte a estrutura de dados de array. 
Logo, para solucionar o problema, a forma mais fácil é usar estruturas de lista que possuem métodos para realizar operações.

Para solucionar o problema do exercicio, optei pelo método reverse que literalmente faz a lista aparecer invertida, além disso, usei o método reversed para fazer com que a lista volte ao ponto inicial.
