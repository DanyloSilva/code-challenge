Neste exercicio é pedido a soma de dois número e o valor desse retorno, ou seja, é necessário criar o input para dois números e após esse input, o programa precisa fazer a soma dos dois.

a = input um número x
b = input um número y

Após o usuário colocar ambos os números, é preciso fazer a soma dos mesmos

Usei o sum para realizar o cálculo:
sum(a, b)
